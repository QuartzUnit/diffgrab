# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-17

### Added
- Web change tracking with unified diff and section-level analysis
- SQLite-based snapshot storage for history management
- markgrab + snapgrab integration for content extraction and screenshots
- 5 MCP tools for agent-driven change monitoring
- 89 tests with full coverage
